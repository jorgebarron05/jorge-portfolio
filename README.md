# Jorge Barron Rivero — Portfolio

Personal portfolio site built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step.

## Structure

```
├── index.html          # Main page
├── resume/             # Resume page + LaTeX source
├── projects/
│   ├── floodit/        # Flood It game (BFS, Canvas)
│   └── maze/           # Maze generator (Kruskal's MST, BFS/DFS)
├── css/
│   ├── base.css
│   ├── sections.css
│   └── maps.css
└── js/
    ├── ui.js
    └── maps.js
```

## Features

- Interactive world map and NYC map (Leaflet.js)
- Particle canvas in the hero
- Scroll-reveal animations
- Playable Flood It and Maze Generator projects
