
# Portfolio Website for Jorge Barron

This is a minimalist, professional portfolio website designed to showcase my skills, experience, and projects.

## Technologies Used
- HTML5 & CSS3
- GitHub Pages for hosting

## Structure
- `index.html`: Main content of the website.
- `styles.css`: Styling for a clean, modern look.
- `script.js`: Placeholder for future interactivity.

## Setup
1. Clone the repository.
2. Open `index.html` in a browser, or deploy to GitHub Pages.

---

This site is a digital version of my resume, with sections on Education, Skills, Work Experience, Projects, and more.
